Common Questions

Is the CLI really free forever?

Yes! The IndieLog CLI is free forever with no limitations. All core features are included permanently.

Do I need Pro to use IndieLog?

No! The CLI is completely free. Pro dashboard is optional for teams that want web-based management and analytics.

When is Pro dashboard launching?

Pro dashboard is coming soon. Early access available for supporters. Follow updates on GitHub for launch details.

Can I use CLI for commercial projects?

Absolutely! The free CLI is perfect for commercial projects, indie companies, and enterprises.

Is there an open source license?

Yes! IndieLog is open source under the MIT License. You can self-host, modify, and contribute.

What about data privacy?

With the CLI, you control everything locally. No data sent to our servers unless you opt-in to Pro dashboard.
